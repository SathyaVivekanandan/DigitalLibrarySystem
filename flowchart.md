# Flowchart and Algorithm
## Application Flowchart

Start  
↓  
Display Main Menu  
↓  
Read User Choice  
↓  
Check Selected Option  
↓  
Add Book / View Books / Search Book / Issue Book / Return Book  
↓  
Display Result  
↓  
Return to Main Menu  
↓  
Exit  
↓  
Stop  

## Algorithm

1. Start the application.
2. Create a Library object.
3. Display the main menu.
4. Get the user's choice using Scanner.
5. Based on the choice, perform the required operation.
6. Display the result message.
7. Repeat the process until the user selects Exit.
8. Stop the application.

## Modules Identified

### Module 1: Book Management
- Add a new book
- View all books
- Search books by title or author
- Display book availability

### Module 2: User Management
- Store user details
- Identify the user who issues a book

### Module 3: Issue and Return Management
- Issue an available book
- Return an issued book
- Update book availability status
