# DigitalLibrarySystem
A console-based Digital Library System developed using Java.
## Features
- Add a new book
- View all available books
- Search for a book
- Issue a book
- Return a book
- Display book availability status
## Technologies Used
- Java
- Object-Oriented Programming (OOP)
- ArrayList
- Scanner class
